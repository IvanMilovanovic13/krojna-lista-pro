# Krojna Lista PRO

AI-assisted kitchen design & cutlist automation tool.

This project is focused on simplifying real-world furniture production workflows by automating:

• kitchen element positioning  
• cutlist generation  
• technical specifications  
• material calculations  

The solution is designed for practical use by furniture workshops and non-technical users.

Key focus:

• real production logic (EU standards)
• reduction of manual calculations
• AI-assisted workflow structuring
• modular architecture

Tech stack:

Python  
Streamlit  
Workflow automation logic  

Purpose:

Translate real carpentry and presales logic into structured digital solutions.
